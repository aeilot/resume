# Resume

This is the LaTeX resume of Chenluo Deng. Last Update: 20 Aug 2025.

## Overview

A clean, professional resume template written in LaTeX using XeLaTeX for compilation.

## Usage

To compile the resume:

```bash
xelatex main.tex
```

## Requirements

- XeLaTeX
- Required packages: `latexsym`, `fullpage`, `titlesec`, `marvosym`, `color`, `verbatim`, `enumitem`, `hyperref`, `fancyhdr`

## Structure

- **main.tex** - Main LaTeX source file containing the resume content
- Clean, single-column layout
- Sections: Education, Experience, Projects, Programming Skills
